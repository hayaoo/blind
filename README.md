# Blind

> 目を閉じて、今を確認する。

macOSメニューバーに常駐するマインドフルネスリマインダーアプリ。

## 概要

フロー状態で働いていると、気づかないうちに時間が過ぎ、疲れにも迷いにも気づかない。Blindは「目を閉じる」という身体的アクションを通じて、「今、正しいことをしているか？」を確認する機会を作る。

## 特徴

- **メニューバー常駐** - 邪魔にならずにいつでもアクセス可能
- **カメラで目を検知** - 目を閉じたことを確認して完了
- **5秒で完結** - 短時間で意図的な中断を作る

## 動作環境

- macOS 14.0 (Sonoma) 以上
- FaceTimeカメラ搭載Mac

## 使い方

1. アプリを起動するとメニューバーにアイコンが表示される
2. 30分ごとに通知でリマインド、またはアイコンクリックで手動起動
3. 画面が表示されたら目を閉じる
4. 5秒間目を閉じるとセッション完了

## 開発

```bash
# リポジトリをクローン
git clone https://github.com/hayaoo/blind.git
cd blind

# Xcodeで開く
open Blind.xcodeproj
```

### 技術スタック

- Swift 5.9+
- SwiftUI / AppKit
- Vision framework（目検知）
- AVFoundation（カメラ制御）

## プライバシー

- カメラ映像は保存しません
- 全ての処理はデバイス上で完結します
- 映像を外部に送信することはありません
